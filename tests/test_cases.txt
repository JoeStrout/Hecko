# Hecko command parse test cases
#
# Format:
#   > user input text
#   module: expected_module (or "none" for no match)
#   command: expected_command
#   arg_name: expected_value
#   arg_name.attr: expected_value   (dotted attribute access)
#   score: expected_score            (check Parse.score)
#   arg_name: nonempty               (list has >= 1 element)
#
# Values are auto-typed: true/false -> bool, none -> None,
# integers, floats, else string.
#
# Every arg in the parse result must be mentioned in the test case.
# If the parse has an arg you didn't list, the test fails.
#
# Lines starting with # are comments. Blank lines are ignored.
